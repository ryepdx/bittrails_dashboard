Add a Custom Datastream

To add a custom datastream, you must put a text file containing your data on a publicly accessible server. Each line of the text file must consist of the date a datum was recorded, in ISO 8601 format (i.e., "YYYY-MM-DD"), followed by a comma, followed by the datum.

For example: 

2012-12-31,100
2013-01-01,89
2013-01-02,94
2013-01-03,101

We will redownload the file once a day and refresh your data on Bit Trails using it. Removing data from the file will not remove them from Bit Trails, but changing a datum associated with a date in the file will change that datum on Bit Trails.

Currently only numeric data is permissible.
